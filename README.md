# We-make-calculations-for-the-customer
Вы работаете в компании «Мегасеть» — это федеральный оператор сотовой связи. В базовый пакет услуг компании входят телефонные звонки, интернет-трафик и сообщения.

С каждым годом клиенты всё активнее пользуются услугами связи, растёт количество звонков и объём интернет-трафика. Коллеги из продуктовой команды хотят проанализировать текущие тарифные планы компании и активность клиентов, а также оценить долю клиентов, которые пользуются услугами сверх тарифного плана. Это позволит обновить линейку тарифных планов под запросы клиентов, а также разработать систему специальных предложений для самых активных клиентов.

«Мегасеть» предлагает клиентам два тарифных плана: Smart и Ultra. Они различаются ежемесячным платежом и пакетом включённых услуг:
Тариф	Ежемесячный платёж	Длительность звонков в минутах	Объём интернет-трафика в гигабайтах	Количество сообщений
Smart	550 руб.	500	15	50
Ultra	1950 руб.	3000	30	1000

Различается и стоимость услуг сверх тарифного плана:
Тариф	Минута разговора	Гигабайт интернет-трафика	Сообщение
Smart	3 руб.	200 руб.	3 руб.
Ultra	1 руб.	150 руб.	1 руб.

Счёт выставляется клиенту в начале каждого календарного месяца. Объём услуг, то есть длительность разговоров, объём интернет-трафика и количество сообщений, считается за полный календарный месяц для каждого клиента. При этом длительность звонков округляется вверх.

Вам поступила задача от команды продукта, которая занимается развитием линейки тарифных планов. Нужно получить информацию о том, как клиенты пользуются услугами компании с точки зрения двух тарифных планов:
1. Выгрузить данные клиентов с информацией о ежемесячных объёмах услуг (суммарная длительность звонков, объём интернет-трафика, количество сообщений) и посчитать текущие траты клиентов — согласно тарифным планам.
2. Для каждого тарифного плана рассчитать среднее значение трат клиентов на услуги связи. При этом нужно учитывать только тех клиентов, которые продолжают пользоваться услугами компании.
3. Среди действующих активных клиентов найти тех, кто пользуется услугами сверх тарифа, и посчитать их средние расходы и среднее значение переплаты для каждого тарифа.

Используйте базу данных, которая содержит информацию об активности клиентов. Подготовьте данные для коллег и проведите необходимые расчёты.

База данных состоит из нескольких таблиц схемы telecom: users, calls, internet, messages и tariffs.

Таблица users
Содержит информацию о пользователях.
- user_id	Идентификатор пользователя. Первичный ключ
- age	Возраст пользователя
- churn_date	Дата окончания договора. Если договор действующий, то поле содержит пустую строку
- city	Город проживания пользователя
- first_name	Имя клиента
- last_name	Фамилия клиента
- reg_date	Дата регистрации пользователя
- tariff	Тариф пользователя. Внешний ключ, связан с первичным ключом tariff_name таблицы tariffs

Таблица calls
Содержит информацию о звонках.

- id	Идентификатор звонка. Первичный ключ
- call_date	Дата звонка
- duration	Длительность звонка в минутах
- user_id	Идентификатор пользователя. Внешний ключ, связан с первичным ключом
- user_id таблицы users

Таблица internet
Содержит информацию о длительности интернет-сессий.
- id	Идентификатор сессии. Первичный ключ
- mb_used	Количество мегабайтов интернет-трафика, потраченного за сессию
- session_date	Дата сессии
- user_id	Идентификатор пользователя. Внешний ключ, связан с первичным ключом
- user_id таблицы users

Таблица messages
Содержит информацию об отправленных сообщениях.
- id	Идентификатор сообщения. Первичный ключ
- message_date	Дата сообщения
- user_id	Идентификатор пользователя. Внешний ключ, связан с первичным ключом
- user_id таблицы users

Таблица tariffs
Содержит информацию о тарифных планах.
- tariff_name	Название тарифа. Первичный ключ
- messages_included	Число бесплатных сообщений, которые входят в тариф
- mb_per_month_included	Количество мегабайтов интернет-трафика, которое входит в тариф
- minutes_included	Число минут разговора, которое входит в тариф
- rub_monthly_fee	Месячная абонентская плата в рублях
- rub_per_gb	Стоимость дополнительного гигабайта интернет-трафика сверх тарифного пакета в рублях (1 гигабайт = 1024 мегабайта)
- rub_per_message	Стоимость отправки сообщения сверх тарифного пакета в рублях
- rub_per_minute	Стоимость минуты разговора сверх тарифного пакета в рублях
